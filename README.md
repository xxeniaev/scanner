# Сканер TCP и UDP портов

### Описание
* Пользователь вводит доменное имя или IP адрес. 
* Осуществляется поиск открытых портов
* Найденные открытые порты выводятся на консоль

### Использование 

`python3 main.py 94.130.99.91`
